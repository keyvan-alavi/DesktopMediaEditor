# Media Editor – Python & PyQt6

![Main Screenshot](assets/screenshot/main.png)

## 📌 Overview
Media Editor is a modern desktop application built with **Python** and **PyQt6** for editing images and videos.  
It offers a clean, bilingual interface (English & Persian) with features such as:
- Image filters and quality adjustments
- Video trimming and audio extraction (MP3)
- Multiple themes and dark mode
- Persistent settings with QSettings
- Responsive, user-friendly design

---

## 🚀 Features
- **Image Editing**: Apply filters (Blur, Sepia, Grayscale, etc.), adjust brightness/contrast, rotate, resize.
- **Video Editing**: Trim videos by time range, extract audio to MP3.
- **Themes**: Default, Dark, Blue, Green.
- **Settings**: Save theme, font size, and dark mode preferences.
- **Cache Management**: Automatic cleanup on exit.

---

## 📂 Installation
```bash
# Clone the repository
git clone https://github.com/keyvan-alavi/DesktopMediaEditor.git
cd MediaEditor

# Install dependencies
pip install -r requirements.txt

# Run the application
python main.py
```

---

## 🛠 Requirements
- Python 3.9+
- PyQt6
- Pillow
- MoviePy
- openCv2

ویرایشگر رسانه یک نرم‌افزار دسکتاپ مدرن است که با **Python** و **PyQt6** ساخته شده و برای ویرایش تصویر و ویدئو طراحی شده است.
